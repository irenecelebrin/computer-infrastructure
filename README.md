Project for the course 'Computer Infreastructure', ay 2025/2026, Higher Diploma in Computing for Data Analytics, ATU Galway Mayo. 


## 1. Data from yfinance
The python package yfinance is used to pull data from the website Yahoo Finance. Although not officially affiliated with the website, Yfinance provides an API that allows to download different types of data from the website (documentation: https://ranaroussi.github.io/yfinance/reference/index.html.) 
A function is created in the jupiter notebook to pull hourly data for the FAANG stocks.
